About Me

I started learning Pascal when i started college, before learning Visual Basic, Java and C++. Adding to my skill set was Python which I learnt and taught during my PGCE. While teaching HE learners on HND i learnt alonmg side the class Visual C#. Completing my Masters i have gained experience using SQL, R and more advanced programming knowledge using Python.

One of the area's I enjoy teahcing is Web Development, which I do with the Level 1, and Level 2 learners. I also support our learners that take part in the World Skill Competition at the Welsh level with Web Development, allowing the opportunity to help develop the base skills with Level 3 learners. I am taking part in this course to further my own knowledge with Web Development so that I can further aid those taking part in competitions, as well as further continuing my own development in the ever change world of Web Developmehnt.
